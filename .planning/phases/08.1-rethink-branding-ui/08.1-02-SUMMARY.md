## Decisions Made

- Card shadow removed in dark mode (dark:shadow-none) since dark backgrounds make shadows invisible and create artifacts
- Dialog/AlertDialog elevated from bg-background to bg-card to create visual hierarchy between page content and modal overlay
- Skeleton changed from bg-accent to bg-muted as muted provides better visual feedback for loading states in the new palette
- Sonner toasts styled via toastOptions classNames instead of only CSS variable overrides for more explicit control
- Six files (button, badge, tabs, input, textarea, tooltip) required no changes as they already used semantic tokens correctly
